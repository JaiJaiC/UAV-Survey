# XAG（极飞科技）

## 一句话定位

以农业无人机为核心、向无人车与数字农业延伸的低空作业系统公司。其竞争力来自高负载飞行、精准施用、农田数据和服务网络的结合。

## 产品与系统栈

- P 系列农业无人机：喷洒、播撒等可换任务系统。
- RevoSpray/RevoCast：液体喷洒与颗粒播撒模块。
- 农业无人车、遥感/测绘设备、自动驾驶控制与农场管理平台。
- RTK/导航、作业规划、地块建图、变量作业和作业记录形成数据闭环。

## 核心高级能力

- **重载与流场控制**：农业效果不是“飞到地块上方”即可，雾滴粒径、下洗气流、冠层穿透和漂移控制共同决定效果。
- **仿地与避障**：田埂、电线、树木和坡地要求雷达/视觉/定位融合；可靠性比实验室识别率更重要。
- **变量农业**：把遥感处方图转成每一区域的喷洒/播撒参数，是从机械替代走向农艺增益的关键。
- **作业基础设施**：快充、换电、补给、运输、培训与售后决定旺季吞吐量。

## 关键判断与风险

- P150 等产品的最大起飞重量、流量和效率均为特定配置/工况下的官方值，跨品牌比较必须统一载荷、亩均用量、地块形状和补给时间。
- 农药登记、无人机施药规范、飞手要求和数据跨境规则因地区不同。
- 商业壁垒很大部分在渠道与服务，而非单一飞行器。

## 建议追踪指标

- 每小时有效作业面积（包含补液/换电/转场）、亩均能耗与漏喷/重喷率。
- 复杂地形任务中断率、旺季故障修复时间和电池循环寿命。
- 变量施用相对均匀施用的投入节省、产量和环境收益。
