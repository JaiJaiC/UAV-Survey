# EHang（亿航智能）

## 一句话定位

无人驾驶载人 eVTOL 的适航和商业运营先行者。它的研究价值主要在“航空器 + 指挥控制 + 适航 + 运营场景”整体闭环，而不是传统小型无人机性能对比。

## 核心产品与里程碑

- EH216-S：双座、无人驾驶、多旋翼 eVTOL，目标场景包括文旅观光和短途城市空中交通。
- 官方与中国民航局体系下的关键证照链：2023 年型号合格证（TC）与标准适航证（AC），2024 年生产许可证（PC），2025 年首批相关运营人取得运营合格证（OC）。
- 产品线也涉及物流、消防和更长航程构型；不同型号的适航状态必须分开核验。

## 核心高级能力

1. **无人驾驶架构**：把飞行控制、任务规划、冗余系统和地面指挥中心结合，降低机上驾驶员依赖。
2. **分布式电推进**：多旋翼带来机械简洁与悬停冗余思路，但航程、噪声、能耗和载荷效率是固有限制。
3. **适航工程**：官方适航页面披露了结构、材料、环境、电磁兼容、软件、数据链和整机飞行等大量符合性验证，这比演示飞行更具产业意义。
4. **运营落地**：起降点、空域、维修、充电、调度和应急处置共同决定商业可行性。

## 关键判断与风险

- “取得某类证书”不等于可在任意城市、任意航线自由运营；航空器、运营人、人员、航线/空域和起降设施是不同合规层级。
- 需要区分试飞/演示、收费运营、常态化规模运营。
- 最大不确定性是可用率、维护成本、保险、社会接受度、噪声和单位座公里成本。

## 建议追踪指标

- 带证航空器交付量、实际载客航班数、机队利用率与事故/中断信息。
- 单架维护工时、电池寿命、起降点建设成本和运营人盈利模型。
- 海外型号认可进展，及新型号与 EH216-S 适航证照的边界。
