# Skydio

## 一句话定位

以计算机视觉自主飞行为核心的美国无人机公司，重点服务公共安全、国防与工业巡检。值得研究的是“高自主单机 + 自动机场 + 浏览器远程运控”的组合。

## 产品与系统栈

- Skydio X10：多传感器行业平台，强调视觉导航、近距巡检和公共安全任务。
- Dock for X10：自动充电、防护、远程起降与预飞检查。
- Remote Ops / DFR Command：浏览器端机队管理、任务执行和 Drone as First Responder 工作流。
- 3D Scan 等软件把自主航迹生成与资产数字化结合。

## 核心高级能力

- **视觉自主**：多摄像头感知、实时三维环境理解和路径规划，目标是降低飞手在复杂近障环境中的操作负担。
- **远程运营**：Dock 官方资料强调快速出动、蜂窝连接、天气/空域感知以及多机场部署。
- **任务级软件**：价值从避障延伸到重复巡检、证据管理、调度和跨部门协作。
- **合规定位**：美国公共安全和政府采购环境既是市场壁垒，也使其产品路线受本土合规要求强影响。

## 关键判断与风险

- 自主能力必须用细线缆、弱纹理、反光/透明物、低照、雨雪等边界场景测试。
- DFR 的规模化依赖 BVLOS 许可、远程驾驶员配置、社区隐私政策和证据治理。
- 软件订阅和封闭生态可提升持续收入，也可能增加客户迁移成本。

## 建议追踪指标

- 每名远程驾驶员可监管的机队规模、自动任务完成率与人工接管率。
- Dock 出动时间、可用率、网络失联恢复和恶劣天气停飞比例。
- 3D Scan 的采集完整性、精度、处理时延和第三方系统集成。
