# Quantum Systems

## 一句话定位

德国电动 VTOL 固定翼与空中情报公司，兼具测绘和国防双用途能力；Vector/Vector AI 体现了便携式平台、边缘 AI 和抗干扰任务系统的融合。

## 产品与系统栈

- Trinity Pro：面向测绘和大范围数据采集的电动 VTOL 固定翼。
- Vector / Vector AI：面向 ISR 的便携式 VTOL 固定翼。
- Scorpion：与 Vector 体系相关的多旋翼构型思路，强调任务适配。
- 任务载荷、地面控制、数据链和 AI 分析构成端到端“aerial intelligence”。

## 核心高级能力

- **VTOL + 固定翼**：无需跑道并保留巡航效率，适合分散部署。
- **边缘 AI**：Vector AI 官方发布强调机载目标检测/跟踪、任务自主和实时态势感知。
- **抗干扰设计**：面向争用环境的导航与链路韧性是重要方向，但公开宣传不能替代独立测试。
- **模块化开放架构**：MOSA 思路方便更换载荷、数据链和计算模块，降低版本演进成本。

## 关键判断与风险

- 双用途平台的出口管制、最终用户限制和供应链合规影响市场可达性。
- 战场部署形成的反馈可能加速迭代，但性能细节常不公开，研究中应区分证实信息和公司叙述。
- VTOL 固定翼仍受转换段风险、侧风和回收区条件影响。

## 建议追踪指标

- 实载航时、部署时间、链路受扰下任务完成率和边缘识别时延。
- Vector AI 的量产、客户部署、载荷兼容和软件更新节奏。
- Trinity 的测绘精度、每平方公里成本和合规认证。
