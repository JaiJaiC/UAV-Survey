# 5. UAV Domain Surveys

Broad surveys of specific directions within UAV/Aerial Robotics research.

## Survey Topics

| Topic | Folder | Description |
|-------|--------|-------------|
| Anti-UAV | `Anti-UAV/` | Counter-drone technologies and methods |
| Swarm Intelligence | `Swarm Intelligence/` | Multi-agent swarm algorithms and coordination |

## Purpose
These surveys provide breadth — understanding the landscape of each sub-field before diving deep into specific papers (→ 1.Paper).
