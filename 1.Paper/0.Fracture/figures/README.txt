=== 论文截图使用说明 ===

1. 从论文 PDF 中截取重要图片（Win+Shift+S 截图）
2. 保存到此目录下，建议命名：
   - fig_arch.png     → 系统架构图
   - fig_network.png  → 网络结构图
   - fig_result.png   → 实验结果/曲线图
   - fig_method.png   → 方法原理图
   或者保留原论文图号：fig1_system.png, fig3_training.png 等

3. 在 note.html 的论文原图解读区（Section 6），把 <img src="figures/xxx.png"> 的
   src 改成对应文件名，onerror 占位符会自动替换

4. 每张图下方填写图解（图展示了什么、各模块含义、关键设计）
