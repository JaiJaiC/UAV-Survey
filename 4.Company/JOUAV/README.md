# JOUAV（纵横股份）

## 一句话定位

中国工业级垂直起降固定翼代表厂商，优势场景是大范围测绘、能源通道巡检、应急与长航时监测。

## 产品与系统栈

- CW 系列 VTOL 固定翼：从中小型电动平台到更大载荷/航程平台。
- 任务载荷：中画幅航测相机、倾斜摄影、LiDAR、EO/IR/激光测距、SAR 等。
- 软件：任务规划、飞行控制、实时视频/地理信息、测绘处理与行业应用。

## 核心高级能力

- VTOL 固定翼把无需跑道与固定翼巡航效率结合，适合狭小起降点和线性基础设施。
- CW-15 官方标称空载航时最高 180 分钟、载荷 3 kg，并提供 RTK 级定位与多任务载荷选项；实际任务需按载荷、风场和安全余量重算。
- 全数字总线、故障恢复、航电扩展和开放架构体现其“行业平台”定位。
- 高海拔、低温、轻雨与复杂电磁环境适应性是工程价值，但应由项目验收数据验证。

## 关键判断与风险

- 与多旋翼相比，优势在覆盖效率和航时；代价是转换飞行复杂度、固定翼巡航避障难度和更高运维要求。
- 项目收入可能依赖行业采购周期，硬件交付后的软件订阅和数据服务能力值得持续观察。
- SAR、LiDAR 等高价值载荷的集成质量、标定和数据处理链比“可挂载”更重要。

## 建议追踪指标

- 实载航时、每平方公里总作业时间、数据精度与返飞率。
- 起降场地要求、部署人数、平均修复时间和任务载荷切换时间。
- BVLOS/适航或行业许可进展、海外服务网络。
