# Autel Robotics（道通智能航空）

## 一句话定位

中国代表性的民用/行业无人机厂商，产品从便携多旋翼延伸到自动机场和倾转/垂直起降固定翼，技术观察重点是多传感器融合、弱 GNSS 自主飞行与行业替代能力。

## 产品与系统栈

- EVO Max：便携式行业多旋翼，面向公共安全、搜救、消防与巡检。
- Autel Alpha：更高等级的行业多旋翼平台。
- Dragonfish：倾转式 VTOL 固定翼家族，面向长航时巡查和 ISR。
- EVO Nest/Dragonfish Nest：远程值守与自动化任务；配套 Autel Enterprise App、云服务及 Mobile/Payload SDK。

## 核心高级能力

- EVO Max 官方资料强调双目视觉 + 毫米波雷达、三维路径规划、弱/无 GNSS 导航及 AI 目标识别。
- A-Mesh 面向机间协同与中继；SkyLink 采用多频段、自适应抗干扰与加密链路。
- 4T 载荷集成长焦、广角、640×512 热红外和激光测距，代表“小型机多传感器一体化”路线。
- Dragonfish 用 VTOL 降低固定翼部署门槛，适合把航时与场地独立性结合起来。

## 关键判断与风险

- 与 DJI 的比较不能只看纸面参数，应重点测飞控稳定性、任务软件成熟度、SDK 文档、售后与机场可用率。
- “无 GNSS”“抗干扰”和识别距离受环境与版本影响很大，需要现场测试。
- 不同地区开放的频段、功能和认证不同，官网标称能力不等于当地可合法启用。

## 建议追踪指标

- EVO Nest 的任务成功率、充电周转、恶劣天气能力与远程故障恢复。
- Autel SDK 的版本节奏、第三方载荷数量、私有云/本地部署支持。
- Dragonfish 实载航时、链路覆盖与部署人员数。
