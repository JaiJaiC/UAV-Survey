# Anduril Industries

## 一句话定位

以 Lattice 指挥控制与自主软件为核心的美国国防科技公司，无人机产品覆盖 Ghost、Altius、Bolt 等；真正的研究对象是软件平台如何连接传感器、无人载具和操作员。

## 产品与系统栈

- Lattice：融合传感器、目标/轨迹、任务分配、操作界面和自主控制的核心平台。
- Ghost：便携、模块化 VTOL 小型无人机，面向 ISR 和多任务载荷。
- Altius：可从空中、地面或海上发射的模块化平台家族。
- Bolt 等小型自主空中系统；更广泛业务还包括反无人机、水下和大型航空器。

## 核心高级能力

- **软件定义系统**：把传感器发现、数据融合、任务分配和平台控制放在共同软件层，而非为每种硬件单独建烟囱。
- **多平台协同**：官方资料强调一名操作员管理多个自主资产；研究价值在任务级人机协同与异常管理。
- **模块化开放系统**：载荷、通信和任务组件可替换，利于快速迭代和供应链替代。
- **产品化交付**：采用更接近商业科技的研发节奏、内部研发投入和批量制造思路进入国防采购。

## 关键判断与风险

- Lattice 的实际开放性、第三方集成成本和数据权属需要通过合同/接口材料而非宣传判断。
- 多机自主的可靠性应关注通信退化、冲突消解、误报和人类监督，而不只是演示规模。
- 业务高度受政府合同、出口管制、政策与伦理约束。

## 建议追踪指标

- Lattice 接入的平台/传感器数量、集成周期和实战/演训可用率。
- 自主系统从原型、试验到量产合同的转化速度。
- 软件收入/持续服务在总体交付中的占比与客户锁定效应。
