# Zipline

## 一句话定位

把无人机当作物流网络而不是单一硬件销售的代表公司，从医疗物资长距离配送扩展到城市家庭即时配送。

## 两代平台逻辑

- Platform 1：固定翼长距离配送，采用发射/回收基础设施与伞投，适合医疗供应链和区域级网络。
- Platform 2：面向家庭和门店的电动 VTOL 配送；主飞行器在较高高度巡航，末端通过独立投递装置精确下放包裹并返回。
- 配套能力包括订单/库存系统集成、机队调度、健康监测、航线规划、充电与运营中心。

## 核心高级能力

1. **高可靠自主运营**：真正壁垒是大量重复商业航班形成的安全、维护、气象和调度数据。
2. **系统解耦**：Platform 2 将巡航飞行与最后几十米的精确、低扰动投递解耦，减少主飞行器贴近人员和障碍物。
3. **网络优化**：库存位置、订单优先级、航线、天气、充电和服务半径共同决定服务水平。
4. **航空运营能力**：监管许可、安全管理系统、维修流程和社区沟通是与飞行器同等重要的资产。

## 关键判断与风险

- 官方披露的累计飞行/配送数字要区分平台、国家、商业与测试口径。
- 单次配送成本高度依赖订单密度、往返距离、有效载荷、天气停飞和基础设施摊销。
- 医疗物流的社会价值不能直接证明普通零售配送的单位经济性。

## 建议追踪指标

- 每站日订单量、准时率、取消率、每单人工介入和每架次成本。
- 机队可用率、部件寿命、天气覆盖、噪声与公众投诉。
- 商业网络从试点到常态化扩区的速度和监管许可范围。
