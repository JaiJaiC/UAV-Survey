# Shield AI

## 一句话定位

以 Hivemind 自主驾驶软件为核心、同时拥有 V-BAT 航空器的美国国防科技公司。研究重点是“AI pilot”如何跨平台迁移及在 GNSS/通信受限环境工作。

## 产品与系统栈

- Hivemind：面向无人航空器和其他平台的任务自主软件，包括感知、规划、协同与执行。
- V-BAT：单发涵道风扇式垂直起降固定翼，面向舰载、边境与持续 ISR。
- Nova 等小型系统及仿真/训练工具；公司亦将 Hivemind 集成到合作伙伴平台。

## 核心高级能力

- **任务自主而非航点自动化**：目标是在不持续遥控的情况下理解任务、重规划并协同。
- **GNSS/通信受限**：依赖机载感知、估计和边缘计算降低持续链路需求。
- **软件跨平台**：如果同一自主栈能快速适配不同动力学、传感器和任务，软件资产可比单一机体更具扩展性。
- **V-BAT 构型**：尾座式 VTOL 减少跑道/弹射回收设备，固定翼巡航提供更长滞空；转换控制和起降环境是技术难点。

## 关键判断与风险

- “自主”必须拆成导航、感知、任务决策、协同和武器授权等不同等级，不能用一个词概括。
- 官方战场表述难以独立复核；应关注可审计测试、客户验收和安全约束。
- 国防采购周期、出口管制、模型验证和责任边界会限制部署速度。

## 建议追踪指标

- 单操作员可管理的平台数、失联任务完成率、任务重规划时间。
- Hivemind 新平台适配周期、仿真到实机差距、客户项目从演示到列装的转化。
- V-BAT 实际出动率、舰面回收表现、维护工时与载荷升级。
