# Parrot

## 一句话定位

法国专业无人机厂商，聚焦测绘、巡检和政府市场；在 4G 主链路、开放软件栈、网络安全与摄影测量集成方面具有代表性。

## 产品与系统栈

- ANAFI Ai：48 MP、4K 影像、可转向立体视觉避障、4G/Wi-Fi 链路和摄影测量任务。
- ANAFI USA/政府产品线：强调安全、热成像和受监管客户。
- FreeFlight 7、OpenFlight、Ground SDK/Air SDK 与 PIX4D 生态。

## 核心高级能力

- **蜂窝数据链**：ANAFI Ai 官方资料称可在 Wi-Fi 与 4G 间自动切换，支持 1080p 直播和飞行中上传；BVLOS 是否可用仍取决于当地批准。
- **开放机器人平台**：飞控应用和 SDK 的开放程度适合科研及行业定制，可访问传感器、占据栅格和网络能力。
- **摄影测量闭环**：网格、双网格、环绕和一键建筑任务可直接与 PIX4D 工作流衔接。
- **安全架构**：安全元件、链路加密与软件完整性是政府/关键基础设施采购的重要差异点。

## 关键判断与风险

- ANAFI Ai 发布时间较早，应持续核查硬件更新节奏、供应可得性和软件维护周期。
- 4G 消除了专用图传距离瓶颈，但引入运营商覆盖、时延、资费、网络拥塞和数据治理依赖。
- 摄影测量精度需在相同 GSD、控制点、快门和风场条件下比较。

## 建议追踪指标

- SDK 活跃度、固件更新、安全公告和第三方应用数量。
- 4G/Wi-Fi 切换稳定性、端到端时延与失联处置。
- 大范围测绘的影像有效率、模型精度和云端处理耗时。
