# DJI（大疆创新）

## 一句话定位

全球民用无人机产业中最完整的“飞行器 + 载荷 + 图传 + 地面站 + 云平台 + SDK”体系之一；研究重点不只是单机性能，而是其模块复用、供应链规模和开发者生态。

## 产品与系统栈

- 消费/影像：Mini、Air、Mavic、Avata、Inspire 等系列，核心是影像、云台和低门槛飞控体验。
- 行业平台：Matrice 4、Matrice 4D/4TD、Matrice 350 RTK 等，覆盖测绘、消防、能源巡检和公共安全。
- 自动机场：Dock 3 + Matrice 4D/4TD + FlightHub 2，支持固定部署和车载部署，形成远程任务闭环。
- 载荷与数据：Zenmuse 相机/LiDAR、Terra 重建、FlightHub 2 运控，以及 Mobile/UX/Onboard/Payload/Cloud API。

## 核心高级能力

1. **平台化复用**：飞控、云台、视觉感知、OcuSync 图传、电池管理跨多条产品线复用，缩短迭代周期。
2. **多传感器任务闭环**：可见光、长焦、热红外、激光测距、LiDAR 与 RTK 共同覆盖“发现—定位—采集—建模”。
3. **无人值守运营**：Dock 3 官方资料显示可支持 24/7 远程任务、车载部署、多机场任务和外接边缘计算模块；应重点研究任务可用率而非单次航时。
4. **生态壁垒**：第三方载荷、应用开发与云端集成降低行业客户二次开发成本。

## 关键判断

- 最大壁垒是系统工程和规模制造，而非某个单点算法。
- 行业竞争正从“卖飞机”转向“卖数据采集与运维工作流”；Dock/FlightHub/Terra 是观察这一转型的窗口。
- 风险集中在不同国家的采购限制、数据治理、频谱与供应链合规；区域市场可获得性不能按全球官网一概而论。

## 建议追踪指标

- Dock 单站可用率、任务间隔、远程接管率和每任务人工分钟数。
- SDK 兼容范围、第三方载荷数量、私有化部署和数据安全能力。
- LiDAR/摄影测量从采集到成果交付的总成本与精度。
- 各区域认证、采购限制和售后网络变化。
