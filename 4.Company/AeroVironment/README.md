# AeroVironment（AV）

## 一句话定位

美国老牌无人系统公司，产品覆盖手抛式小型侦察无人机、Group 3 长航时 VTOL、巡飞系统及多域机器人，优势在成熟项目、任务载荷与政府客户体系。

## 产品与系统栈

- Puma、Raven、Wasp 等小型 ISR 平台。
- JUMP 20 / JUMP 20-X：长航时 VTOL 固定翼和多传感器任务平台。
- Switchblade 系列巡飞系统；另有地面、海上与反无人机业务。
- 共用地面控制、数据链、任务载荷和软件分析，形成跨平台组合。

## 核心高级能力

- **产品谱系与任务复用**：不同尺寸平台共享客户训练、链路和任务工作流，降低系统级部署摩擦。
- **长航时 VTOL**：JUMP 20 官方规格为 13+ 小时航时、185 km 任务范围和约 13.6 kg 载荷，兼顾无跑道部署与持续 ISR；具体版本和任务剖面需逐项核查。
- **模块化载荷**：EO/IR、雷达、通信等载荷决定任务价值，平台只是载体。
- **体系集成**：侦察、目标信息与其他效应器/指挥系统之间的低时延数据流是重要壁垒。

## 关键判断与风险

- 合并、收购和产品重命名会改变组合边界，历史资料不一定反映当前配置。
- 军用标称“range”可能指链路、作战半径或总航程，不能混用。
- 政府合同集中、预算周期、出口许可和项目竞标影响业绩可见性。

## 建议追踪指标

- 在手订单、项目续约/量产、交付能力和并购整合。
- 实际任务可用率、载荷集成周期、单任务人员与维护负担。
- 小型平台到 Group 3 平台的控制/数据互操作性。
